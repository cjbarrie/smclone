# smclone

Repository for React social media clone environment

![](clone.png)
![](clonetimeline.png)

To run backend:

```
cd smclone/backend
workon smclone
export FLASK_APP=main.py  # For macOS and Linux
flask run --port 5001
```

To run frontend:

```
cd smclone/frontend
workon smclone
npm start
```
